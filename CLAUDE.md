# Прототип — Филоненко Групп

Лендинг юрфирмы (Владивосток, основатель — Татьяна Филоненко). Статика: HTML + CSS + ванильный JS, без сборки. Оригинал, от которого отталкивались: `dev1.swat.one/prototype/filonenko/`.

## Файлы

- `index.html` — светлая версия (первая итерация)
- `dark.html` — **тёмная тема, основная рабочая версия**: хиро с фоновым видео, стек кейсов при скролле, акцентная панель формы, футер с картой
- `hero-v2.html` — песочница варианта хиро (шапка поверх фона)
- `cases-variants.html` — 5 вариантов вёрстки блока кейсов (референс-страница)
- `style.css` — общая база; версии переопределяют её инлайновым `<style>` в `<head>`
- `fonts/` — Onest (фолбэк) и `fonts/Fav/` — Favorit Pro Book/Medium
- `img/` — фото основателя; `video/` — фоновые видео хиро

## Дизайн-система

- **Шрифт:** Favorit Pro — Book (400) для текста, Medium (500) для заголовков; фолбэк Onest
- **Крупные заголовки** — капсом Medium; пункты услуг в тёмной теме — строчными, Book
- **Светлая тема:** фон `#F2F1EE`, текст `#1A1A1A`, акцент `#5A0012` (бордовый)
- **Тёмная тема:** фон `#141414`, панели `#191919`, карточки `#1F1F1F`, текст `#F2F1EE`, акцент `#C9B99D` (беж). Акцент дозированно: лейблы-«брови», номера, выделения слов; блок «Частным лицам» и панель формы — целиком акцентные
- **Скругления:** карточки 16px, панели 8px, чипы и кнопки 7px. Круглые только маркеры/аватарки/соцкнопки футера
- **Кнопки (dark):** два размера — large 64px (паддинг 0 40px, 14px) и small 48px (базовый `.btn`, 13px); три стиля — золотая (`btn--dark` → акцент), белая (`btn--light`), обводка (`btn--outline`/`btn--ghost`)
- Лейблы над заголовками: маленький капс акцентным цветом, без скобок и точек
- Десктоп-прототип: `min-width: 1440px`, тянется шире, `noindex`
- **Макеты из Figma переносим 1:1 по пикселям, БЕЗ масштабирования.** Макс рисует на фрейме 1920, но размеры шрифтов/отступов совпадают с уже свёрстанным (`.display` 84px, `display--md` 52px). Однажды отмасштабировал ×0.75 — пришлось всё откатывать

## Железные правила

- **Иконки не рисуем.** Берём из оригинального сайта (стрелки, плюсы, лупа, Telegram/WhatsApp/Max уже вытащены) или официальные брендовые. Нет иконки — спросить у Макса
- **Коммиты по команде** («сохрани» / «запушь»), пачками. Push в `main` → Vercel деплоит сам
- **Самопроверку не гонять:** Макс сам смотрит превью и говорит, что не так. Скриншот-циклы не устраивать
